Displays post analytics like word count, character count and read time.

== Description ==

Adds a section to the current post with analytics in the preferred location, can be changed in the plugin settings.

Features:
1. Word Count
2. Character Count
3. Read Time

The plugin's settings page can be found in the tools menu.

== Frequently Asked Questions ==

= If all options are disabled, the analytics section will be still shown? =

No, it will be only shown if there is at least one option enabled.
